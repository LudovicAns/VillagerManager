## Key Features

* Block villager reproduction.
* Manage villager trades.
  * Turn on/off trades.
  * Blacklist enchanted book and materials

## How To Use

For now, only one command is available to reload your configurations.

```
/vm reload
```
```
/vm reload <file>
```
